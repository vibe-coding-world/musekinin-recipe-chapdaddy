# 06 最終表示確認

## 目的

画像込みで新規レシピの最終表示を確認する。

## 入力

```text
partials/details/detail_レシピID.html
data/recipe-details.json
data/recipes.json
assets/images/
```

## 依頼文

```text
新規レシピの最終表示確認をして。

確認内容:
- HTTPサーバーを起動する。
- list.html に新規レシピが出る。
- detail.html?id=レシピID が表示される。
- hero画像が表示される。
- step画像が5枚表示される。
- ブラウザコンソールに重大エラーがない。

必要なら修正して。
```

## 起動

```bash
python3 -m http.server 8000
```

## URL

```text
http://127.0.0.1:8000/list.html
http://127.0.0.1:8000/detail.html?id=レシピID
```
