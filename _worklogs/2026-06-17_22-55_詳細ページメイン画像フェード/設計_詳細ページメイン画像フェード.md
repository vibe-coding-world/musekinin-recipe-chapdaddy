# 設計 詳細ページメイン画像フェード

## 構成

```mermaid
flowchart TD
  A[detail_chicken_nanban.html] --> B[hero画像]
  A --> C[fade画像レイヤー]
  C --> D[css animation]
  D --> E[3秒ごとに切替]
```

## 基本方針

Swiperは使わない。

HTMLとCSSだけで実装する。

```mermaid
flowchart TD
  A[Simple] --> B[HTMLに画像を明示]
  B --> C[CSS animation]
```

## DOM方針

既存の `.c_recipe-hero__image` を残す。

HTMLにフェード用のラッパーを明示する。

```mermaid
flowchart TD
  A[.c_recipe-hero] --> B[.c_recipe-hero__fade]
  B --> C[img]
  B --> D[img]
  B --> E[img]
  A --> F[.c_recipe-hero__copy]
```

## 画像配置

```mermaid
flowchart TD
  A[hero画像] --> B[1枚目]
  C[作り方画像] --> D[2枚目以降]
  B --> E[.c_recipe-hero__fade]
  D --> E
```

初回検証では `detail_chicken_nanban.html` に画像を直接書く。

## CSS

```mermaid
flowchart TD
  A[hero] --> B[position relative]
  B --> C[fade images absolute]
  C --> D[keyframes]
```

| class | 役割 |
|---|---|
| `.c_recipe-hero__fade` | 画像レイヤーの親 |
| `.c_recipe-hero__fade-image` | フェード対象 |
| `.c_recipe-hero__fade--auto` | 自動切替 |

## Animation

```mermaid
flowchart TD
  A[6枚] --> B[18秒周期]
  B --> C[3秒ごとに次の画像]
  C --> D[opacity切替]
```

| 項目 | 値 |
|---|---|
| 枚数 | 6枚 |
| 周期 | 18秒 |
| 間隔 | 3秒 |
| 表示 | `opacity: 0.82` |

## reduced motion

```mermaid
flowchart TD
  A[prefers-reduced-motion] --> B{reduce?}
  B -->|yes| C[静止]
  B -->|no| D[フェード]
```

## 初回制限

初回は `chicken_nanban` のみ有効化する。

実装確認後、全詳細ページへ広げる。

```mermaid
flowchart TD
  A[test] --> B[chicken_nanban]
  B --> C[OK]
  C --> D[all details]
```

## 注意

- 既存heroコピーを隠さない。
- フェード用画像は `aria-hidden="true"` にする。
- JS初期化に依存しない。
- 全ページ展開時は各詳細HTMLに同じ構造を追加する。
