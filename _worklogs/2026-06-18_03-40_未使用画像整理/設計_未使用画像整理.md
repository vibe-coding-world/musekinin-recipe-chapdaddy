# 設計 画像整理

## 構成

```mermaid
flowchart TD
  A[assets/images] --> B[参照調査]
  B --> C[未使用削除]
  B --> D[利用中PNG]
  D --> E[cwebp変換]
  E --> F[WebP配置]
  F --> G[参照パス更新]
```

## 方針

画像形式はWebPを基本にする。

`about_recipe_note.jpeg` と `icon-chapdaddy.avif` は既存形式を維持する。

## 変換

| 項目 | 内容 |
|---|---|
| 変換元 | `assets/images/*.png` |
| 変換先 | `assets/images/*.webp` |
| ツール | `cwebp` |
| 品質 | `-q 82` |

## 参照更新

```mermaid
flowchart TD
  A[実装ファイル] --> B[.png参照検索]
  B --> C[.webpへ置換]
  C --> D[参照存在確認]
```

| 対象 | 内容 |
|---|---|
| HTML | `index.html` / `list.html` / `detail.html` |
| JS | `js/*.js` |
| JSON | `data/*.json` |
| CSS | `css/*.css` |
