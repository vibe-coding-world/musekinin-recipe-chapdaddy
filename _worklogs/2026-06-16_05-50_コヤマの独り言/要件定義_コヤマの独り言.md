# 要件定義 コヤマの独り言

## 目的

詳細ページの「まとめ」を「コヤマの独り言」にする。

```mermaid
flowchart TD
  A[まとめ] --> B[コヤマの独り言]
  B --> C[料理の余韻]
  B --> D[無責任レシピの考え方]
```

## 対象

| 対象 | 内容 |
|---|---|
| テストページ | `detail.html?id=kakuni` |
| partial | `partials/details/detail_kakuni.html` |
| CSS | `css/recipe-note.css` |
| 背景 | `assets/images/about_recipe_note.jpeg` |
