# 設計 気分で選ぶコンポーネント

## 構成

`mood-section.js` が `<mood-section>` を描画する。

```mermaid
flowchart TD
  A[js/mood-section.js] --> B[customElements]
  B --> C[mood-section]
  C --> D[index.html]
  C --> E[list.html]
  C --> F[detail.html]
```

## ファイル

| 種類 | ファイル |
|---|---|
| JS | `js/mood-section.js` |
| 利用元 | `index.html` |
| 利用元 | `list.html` |
| 利用元 | `detail.html` |
| CSS | 既存CSS |

## HTML配置

```mermaid
flowchart TD
  A[index.html] --> B[hero]
  B --> C[mood-section]
  C --> D[今のおすすめ]
  D --> E[無責任レシピとは]
  E --> F[shop-banner]

  G[list.html / detail.html] --> H[ページ固有コンテンツ]
  H --> I[mood-section]
  I --> J[shop-banner]
```

| ページ | 配置 |
|---|---|
| `index.html` | 既存セクション位置 |
| `list.html` | `<shop-banner>` の直前 |
| `detail.html` | `<shop-banner>` の直前 |

## コンポーネント

```html
<mood-section></mood-section>
```

## 出力HTML

```mermaid
flowchart TD
  A[mood-section] --> B[section]
  B --> C[c_heading]
  B --> D[c_home-mood-grid]
  D --> E[a.c_home-mood]
```

| 要素 | クラス |
|---|---|
| section | `la_section la_stack u_stack_20` |
| heading | `c_heading c_heading--center` |
| grid | `la_grid c_home-mood-grid` |
| card | `c_home-mood` |

## データ

データはJS内に固定で持つ。

```mermaid
flowchart LR
  A[moods] --> B[label]
  A --> C[href]
  A --> D[image]
  A --> E[alt]
  A --> F[note]
```

| キー | 用途 |
|---|---|
| `label` | 表示名 |
| `href` | 遷移先 |
| `image` | 画像 |
| `alt` | 代替テキスト |
| `note` | 補足文 |

## 注意

| 項目 | 内容 |
|---|---|
| 二重表示 | `index.html` の既存HTMLを置換する |
| URL | `list.html?mood=...` を使う |
| CSS | 既存 `.c_home-mood` を維持する |
| 依存 | `<shop-banner>` とは分離する |
