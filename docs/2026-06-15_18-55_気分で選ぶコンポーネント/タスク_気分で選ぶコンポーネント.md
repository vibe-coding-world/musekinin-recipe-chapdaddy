# タスク 気分で選ぶコンポーネント

## 目的

「気分で選ぶ」を3ページで再利用する。

```mermaid
flowchart TD
  A[実装] --> B[JS作成]
  A --> C[index置換]
  A --> D[list追加]
  A --> E[detail追加]
  A --> F[確認]
```

## タスク

| 状態 | 項目 |
|---|---|
| 完了 | `index.html` の既存セクションを確認する |
| 完了 | `js/mood-section.js` を作成する |
| 完了 | `<mood-section>` を定義する |
| 完了 | `index.html` の既存セクションを置換する |
| 完了 | `list.html` に `<mood-section>` を追加する |
| 完了 | `detail.html` に `<mood-section>` を追加する |
| 完了 | 3ページでJSを読み込む |
| 完了 | 3ページのHTML配置を確認する |
| 完了 | 各カードのリンクを確認する |
| 未着手 | ブラウザで表示崩れを確認する |

## 対象ファイル

| 種類 | ファイル |
|---|---|
| 追加 | `js/mood-section.js` |
| 変更 | `index.html` |
| 変更 | `list.html` |
| 変更 | `detail.html` |
